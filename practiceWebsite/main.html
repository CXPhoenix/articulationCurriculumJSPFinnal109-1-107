第一題

請把下方 Hello 改為 well done

hello
使用Chrome的F12找到Hello的Id，接著使用 
document.getElementById().innerHTML
來處理文字。

此題答案: 
document.getElementById("Q1text").innerHTML = "well done";

第二題

請試著把下方紅色的字換成 容易 兩個字

這件事情真的很難想像

跟上題一樣，找到「難」這個字的id，接著使用 
document.getElementById().innerHTML
來處理內容

解答： 
document.getElementById("Q2text").innerHTML = "容易";

第三題

請試著讓按鈕按下後，就讓藍色的字變成 不告訴你咧

告訴你一個秘密


1. 找到按鈕的id，並設置變數 
var btn = document.getElementById()
2. 接下來確認藍色字的id 3. 最後使用指令讓網頁知道按下按鈕後的事情。
btn.onclick = function() {你指定按下按鈕後會發生的事情寫再者裡喔！};


解答： 
var btn = document.getElementById("Q3btn");
btn.onclick = function() {
    document.getElementById("Q3text").innerHTML = "不告訴你咧";
};

第四題

請試著讓按鈕按下後，就讓藍色的字變成 是個變態

告訴你一個秘密


與上一題相同，找到按鈕的id、找到要改的文字的id，最後寫下按下按鈕後的事件即可。

解答： 
var btn = document.getElemnetById("Q4btn");
btn.onclick = function() {
    document.getElementById("Q4text").innerHTML = "是個變態";
};

第五題

請試著讓紅框中的綠字與綠框中的紅字交換(包含顏色一起交換喔!)

這是紅色字喔～

這是綠色字喔～
1. 找到按鈕的id 2. 確認綠色字和紅色字的id 3. 對兩個顏色字各使用一次指令
document.getElementById().style.color
去改變字的顏色

解答：
var btn = document.getElementById("Q5btn");
btn.onclick = function() {
    document.getElementById("Q5textR").style.color = "red";
    document.getElementById("Q5textG").style.color = "green";
};

第六題

請試著按下按鍵讓隱藏起來的文字顯示出來

哈你發現我了

1. 找到隱藏起來的文字與id（利用按下F12來尋找喔～） 2. 找到按鈕並且跟上題一樣寫下按下按鈕後的事件。 3. 要讓文字顯現出來的指令是：
document.getElementById().style.display = "block";
順帶一提，讓文字消失的指令是：
document.getElementById().style.display = "none";


解答：
var btn = document.getElementById("Q6btn")
btn.onclick = function() {
   document.getElementById("Q6div").style.display = "block";
};
